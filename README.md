# Ordinary Differential Equations [![Build Status][travis-img]][travis-url]

The package provides algorithms for integrating systems of [ordinary
differential equations][1].

## [Documentation][doc]

## Subpackages

The package contains two subpackages:

* [dopri](dopri) and
* [rk4](rk4).
